## shadcn/ui：不是组件库，是组件集合

shadcn/ui 的革命性在于它**不是**一个 npm 包。它是一组可复制的组件代码，直接集成到你的项目中。你可以完全控制源代码。

### 核心理念

1. **所有权（Ownership）**
   - 组件源码在你的仓库中
   - 随意修改、扩展、定制
   - 不依赖第三方包版本

2. **可访问性优先**
   - 内置键盘导航、屏幕阅读器支持
   - 基于 Radix UI / Base UI 的无障碍原语
   - WAI-ARIA 标准兼容

3. **Tailwind CSS 深度集成**
   - 使用 `cn()` 合并自定义类名
   - `cva` 管理变体样式
   - 完全响应式设计

### 常用组件

- **Form**：集成 react-hook-form + zod 验证
- **Dialog**：模态对话框，支持嵌套
- **DropdownMenu**：下拉菜单，支持子菜单
- **Table**：数据表格，可排序和筛选
- **Chart**：基于 Recharts 的图表组件

### 定制主题

通过 CSS 变量定义完整的设计系统：
- 颜色：primary、secondary、destructive、muted
- 圆角：全局 `--radius` 变量
- 间距和排版：继承 Tailwind 默认值

> shadcn/ui 让你从"使用组件库"变为"拥有组件代码"。

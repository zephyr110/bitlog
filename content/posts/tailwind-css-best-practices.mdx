## 为什么选择 Tailwind CSS？

Tailwind CSS 的实用优先（utility-first）方法论彻底改变了前端样式的编写方式。它没有预置的组件样式，而是提供了底层的工具类，让你自由组合。

### 设计系统核心

1. **设计令牌（Design Tokens）**
   ```css
   :root {
     --primary: 222.2 47.4% 11.2%;
     --radius: 0.5rem;
   }
   ```
   通过 CSS 变量定义颜色、间距、圆角等基础设计决策。

2. **组件抽象**
   - 使用 `cn()` 工具函数合并类名
   - 通过 `cva`（class-variance-authority）创建组件变体
   - 提取重复的样式模式为可复用的工具类

3. **响应式设计**
   - 移动优先的断点系统：`sm` → `md` → `lg` → `xl`
   - 容器查询配合 Tailwind 实现组件级响应
   - 使用 `grid` 和 `flex` 构建弹性布局

### 暗色模式

```tsx
<div className="bg-white dark:bg-gray-950 text-gray-900 dark:text-gray-50">
  自适应内容
</div>
```

`dark:` 前缀让暗色模式适配变得极其简单。

> Tailwind 不仅是一个 CSS 框架，更是一种设计哲学。

## Hello, World! 👋

Welcome to my blog! This site is built with:

- **Next.js 16** — Static site generation with the App Router
- **shadcn/ui** — Beautiful, accessible UI components
- **Tailwind CSS v4** — Utility-first CSS framework
- **MDX** — Markdown with JSX superpowers

## Why I Built This

I wanted a blog that is:

1. **Fast** — Static pages served from GitHub Pages
2. **Simple** — Content stored as plain Markdown files
3. **Customizable** — Full control over design and functionality
4. **Free** — No hosting costs, no database bills

## What's Next

Stay tuned for more posts about technology, programming, and whatever else catches my interest.

The admin panel gives me full control over content — writing drafts, editing posts, and publishing with a single click. All while keeping the public site blazing fast.
